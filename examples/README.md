# Examples

## Full Demo

[**→ See the full demo**](full-demo/) — a complete scene showcasing annotations, typing, paste, highlight, split view, and focus all in one recording.

## Commands

One folder per command, each with a `scene.pop` script and a rendered `demo.gif`.

## Table of Contents

| Command | Description |
|---|---|
| [Annotate](annotate/) | Display a text overlay on the video frame |
| [Backspace](backspace/) | Delete characters or remove auto-indent levels |
| [Enter](enter/) | Press Enter and trigger Monaco auto-indent |
| [File](file/) | Open a named tab and run commands inside it |
| [Focus](focus/) | Switch the active panel to a given file |
| [Highlight](highlight/) | Highlight a line or a range in the editor |
| [ImportScene](import-scene/) | Compose a recording from multiple scene files |
| [MoveTo](moveto/) | Move the cursor to a specific line |
| [Paste](paste/) | Insert text instantly, preserving exact formatting |
| [Scroll](scroll/) | Scroll the editor to bring a line into view |
| [Select](select/) | Select a range of text by line and column |
| [Sleep](sleep/) | Pause the scene for a given duration |
| [Split / Unsplit](split/) | Open and close a second editor panel |
| [Tab](tab/) | Press Tab for manual indentation |
| [Themes](themes/) | Apply community themes to the Monaco editor |
| [Type](type/) | Type text character by character |
